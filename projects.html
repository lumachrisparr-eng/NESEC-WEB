Create.

Theory without practice is incomplete. At the heart of our platform is a strong belief that the best learning happens when you build real things. Our project library is designed to bridge the gap between academic knowledge and practical application, giving students the opportunity to create, experiment, fail, iterate, and ultimately succeed.

Why Projects Matter

Theory without practice is incomplete.

Employers and academic institutions increasingly value demonstrated skills over credentials alone. Building a portfolio of completed projects shows that you can apply what you know to real situations, and develops skills that are difficult to teach in a traditional classroom — like debugging under pressure, managing complexity, and adapting when things go wrong.

Beyond career benefits, projects are one of the most satisfying ways to learn. There is a unique sense of accomplishment in seeing a circuit come to life, watching your code produce a meaningful output, or deploying a system that solves a real problem. That experience is what keeps engineers curious and motivated throughout their careers.

Beginner Projects

Level 01

Beginner · Programming

Simple Calculator

Build a fully functional calculator application that handles addition, subtraction, multiplication, and division. This project introduces you to user input handling, conditional logic, and function design. It can be implemented as a command-line program or a graphical user interface, depending on your current skill level. As an extension, you can add support for more complex operations such as square roots, exponentiation, and memory functions.

Beginner · Electronics

Basic Circuit Simulation

Use circuit simulation software such as LTspice or Tinkercad to design and test simple electronic circuits. Start with a basic LED circuit, then progress to circuits involving resistors, capacitors, and transistors. This project builds your intuition for electronic behavior and teaches you how to use industry-standard simulation tools before working with physical components.

Beginner · Logic

Number Guessing Game

Create an interactive game where the computer generates a random number and the user attempts to guess it within a limited number of tries. This project reinforces loops, conditionals, random number generation, and user feedback mechanisms — all fundamental programming concepts.

Intermediate Projects

Level 02

Intermediate · Embedded & Networking

IoT Monitoring System

Design and build an Internet of Things (IoT) system that monitors environmental data such as temperature, humidity, and light levels using sensors connected to a microcontroller. The data is transmitted over Wi-Fi to a cloud dashboard where it can be visualized in real time. This project integrates embedded programming, networking, data visualization, and cloud services into one cohesive system — making it an excellent showcase of cross-disciplinary engineering skills

Intermediate · Web Development

Personal Portfolio Website

Build a fully responsive personal portfolio website to showcase your projects, skills, and contact information. This project teaches HTML, CSS, and JavaScript fundamentals along with responsive design, accessibility, and web hosting. Deploy it on GitHub Pages so it is accessible to anyone on the internet.

Intermediate · Security

File Encryption Tool

Develop a command-line tool that encrypts and decrypts files using a symmetric encryption algorithm such as AES. This project introduces cryptography fundamentals, file I/O operations, and the importance of secure software design — giving you an understanding of how digital security works under the hood.

Advanced Projects

Level 03 · PFE / Mémoire

Advanced · Artificial Intelligence

AI-Based System

Design and train a machine learning model to solve a meaningful problem of your choosing — from an image classifier that identifies plant diseases to a recommendation engine for study materials or an NLP system that summarizes technical documents.

You will work through the entire machine learning pipeline: data collection, preprocessing, model selection, training, evaluation, and deployment. Built with Python and libraries such as TensorFlow and scikit-learn.

Advanced · Embedded Systems

Embedded Automation System

Build an embedded automation system that uses a microcontroller to monitor inputs from multiple sensors and automatically control actuators based on predefined logic.

Applications include a smart irrigation controller, an automated greenhouse system, or a motor control unit. This project demands deep understanding of real-time programming, hardware interfacing, power management, and fault tolerance.

Advanced · Operating Systems

Custom Operating System Shell

Implement a basic command-line shell that can interpret and execute user commands, manage processes, and handle input and output redirection.

This project gives you a profound understanding of how operating systems work — including process creation, system calls, memory management, and inter-process communication. It is one of the most intellectually rewarding projects you can undertake as a computer engineering student.
